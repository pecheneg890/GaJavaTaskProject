# Трекинг задач (Spring Boot)

## Запуск проекта

- Режим использования (приложение и бд внутри контейнеров)

```docker-compose -f docker-compose-test.yml up```

- Режим разработки (в контейнере только бд)  

Запуск БД

```docker-compose up```

Запуск приложения (требования java 25)

```./gradlew bootRun```

## Пользователь по умолчанию
При первом запуске приложения создается пользователь по умолчанию в роли ADMIN    
Логин - admin@admin  
Пароль - admin 

## Описание endpoint-ов
swagger доступен по адресу  
http://localhost:8020/swagger-ui/index.html

### Авторизация
- Регистрация пользователя  

Доступно без авторизации. Вводятся данные нового пользователя, в ответ генирируются access и refresh токен  
Access токен необходимо ввести в раздел ```authorize``` swagger для доступа к закрытым ресурсам
Пользователь создается в роли USER

- Вход  

Вводится логин и пароль, в ответ генерируются access и refresh токен

- Обновление access токена

Используется для генерации новой пары токенов. На входе необходимо указать refresh токен

- Смена пароля

Доступна авторизованным пользователям. На входе необходимо указать новый пароль

### Группа задач
(раздел доступен авторизованному пользователю)
- Создание группы задач

Создает группу задач, на входе необходимо указать название, описание, категорию

- Получение своих групп задач

Выводится список групп задач по авторизованному пользователю

- Изменение группы задач
- Удаление группы задач

### Задачи
(раздел доступен авторизованному пользователю, работа возможна только со своими задачами)
- Создание задачи

Создает задачу, на входе необходимо задать название, описание, дедлайн, приоритет, статус, сложность и id группы задач

- Получение списка задач
- Получение задачи по id
- Изменение статуса задачи
- Изменение задачи
- Удаление задачи

### Консоль администратора
(раздел доступен только пользователью с ролью ADMIN)
- Установка заданному пользователю роли администратора
На вход необходимо указать email пользователя
- Список всех пользователей
- Список всех задач
- Задачи заданного пользователя
- Список всех групп задач
- Статистика кол-ва задач в разрезе пользователя и статуса
